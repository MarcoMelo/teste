O que é python?

Python é uma linguagem de programação de alto nível , interpretada, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte.A linguagem foi projetada com a filosofia de enfatizar a importância do esforço do programador sobre o esforço computacional. Prioriza a legibilidade do código sobre a velocidade ou expressividade. Combina uma sintaxe concisa e clara com os recursos poderosos de sua biblioteca padrão e por módulos e frameworks desenvolvidos por terceiros.

O que é SQLite?

SQLite é uma biblioteca em linguagem C que implementa um banco de dados SQL embutido. Programas que usam a biblioteca SQLite podem ter acesso a banco de dados SQL sem executar um processo SGBD separado. SQLite não é uma biblioteca cliente usada para conectar com um grande servidor de banco de dados, mas sim o próprio servidor. A biblioteca SQLite lê e escreve diretamente para e do arquivo do banco de dados no disco.

O que é Django?

Django é um Framelivre e open source framework de aplicações web, escrito em Python, que segue o controlador de visualização de modelo (MVC) padrão de arquitetura. Ele é mantido pelo Django Software Foundation (DSF), uma organização independente estabelecida como uma organização sem fins lucrativos.O objetivo principal do Django é facilitar a criação de sites, banco de dados orientado complexos. Django enfatiza a capacidade de reutilização e "capacidade de conexão" de componentes, o rápido desenvolvimento, eo princípio da não repetir-se.
